# lsrc

Run GNU ls with an rc config file

## Install

```
sh -c "$(curl -fsSL https://githubraw.com/kilna/lsrc/main/install.sh)"
```

## Usage

```
For usage examples see https://github.com/kilna/lsrc
```

## Examples

First, enable in your shell (pick one):

```
$ lsrc profile ~/.bashrc
```
